# Modern QR Generator (PyQt6)

A modern, stylish, and user-friendly **QR Code Generator** built using **Python**, **PyQt6**, **qrcode**, and **Pillow**. Designed with a sleek dark theme and smooth UI components.

---

## Features

* Modern PyQt6 GUI (dark theme)
* Live QR preview while generating
* Text/URL input
* Foreground & background color pickers
* Adjustable size and border
* Save QR as PNG
* Clean, organized code (single-file app)

---

##  Installation

### 1. Install Python

Download from: [https://www.python.org/downloads/](https://www.python.org/downloads/)

### 2. Install Dependencies

Run this in your terminal:

```sh
pip install PyQt6 qrcode Pillow
```

### 3. Download/Clone This Project

If using Git:

```sh
git clone https://github.com/YourUserName/Modern-QR-Generator.git
cd Modern-QR-Generator
```

Or download ZIP from GitHub.

---

##  Run the Application

```sh
python modern_qr_generator.py
```





---

##  Building EXE (Optional)

If you want a Windows .exe:

```sh
pip install pyinstaller
pyinstaller --noconsole --onefile modern_qr_generator.py
```

Your EXE will appear in the `dist/` folder.
